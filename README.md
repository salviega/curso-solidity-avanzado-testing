## Quick start 🏄

Prerequisites: [Node (v20 LTS)](https://nodejs.org/en/download/) and [Git](https://git-scm.com/downloads)

> Clone the repository:

```
git clone https://github.com/salviega/cross-carbon-contracts.git
```

> Install dependencies:

```
yarn
```

> Run the project:

```
yarn chain
```
